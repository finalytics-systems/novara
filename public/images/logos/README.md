# Client Logos

Place company logo images in this directory. The logos will automatically display when available.

## Logo File Names

### GCC Implementations
- `ubm.png` - Univest Business Machines
- `digisec.png` - DigiSec
- `ticom.png` - ticom
- `levene.png` - Levene

### US and Europe Implementations
- `saw.png` - SAW Components
- `flow.png` - Flow
- `cobb.png` - Cobb

### Pakistan Implementations
- `tcl.jpg` - TCL
- `friends.png` - Friends Home
- `hyundai.jpeg` - Hyundai
- `vapes.png` - Vapes Direct
- `mtn.png` - MTN
- `pwc.png` - PWC

## Logo Specifications

- **Format**: PNG, JPG, or SVG (PNG recommended for transparency)
- **Recommended size**: 200-300px width, maintain aspect ratio
- **Background**: Transparent or white background preferred
- **File size**: Keep under 200KB for optimal performance

## How It Works

- If a logo file exists, it will be displayed
- If a logo file doesn't exist, a placeholder with the company's first letter will be shown
- Logos are optimized automatically by Next.js Image component
