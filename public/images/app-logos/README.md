# App Integration Logos

Place app integration logo images in this directory. The logos will automatically display when available.

## Logo File Names

- `paypal.png` - PayPal
- `dropbox.png` - Dropbox
- `razorpay.png` - Razorpay
- `slack.png` - Slack
- `google.png` - Google
- `stripe.png` - Stripe
- `exotel.png` - Exotel
- `woocommerce.png` - WooCommerce
- `plaid.png` - Plaid
- `aws.png` - AWS

## Logo Specifications

- **Format**: PNG, JPG, or SVG (PNG recommended for transparency)
- **Recommended size**: 150-200px width, maintain aspect ratio
- **Background**: Transparent background preferred
- **File size**: Keep under 100KB for optimal performance
- **Aspect ratio**: Most app logos work best with a horizontal/landscape orientation

## How It Works

- If a logo file exists, it will be displayed
- If a logo file doesn't exist, a placeholder with the app name will be shown
- Logos are optimized automatically by Next.js Image component

## Where to Get Logos

You can find official logos from:
- Company brand guidelines pages
- Logo download sites (ensure you have permission to use)
- Official partner/reseller resources
